HTML

  • The head element contains information about the webpage.
  • The head element is a container for metadata
  • Meta elements contain information about the page that is used by the browser
  • The user wont see information in the head or meta elements
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • To add color you can use HTML color codes
  • You must link your style.css with your main code

Git

  • Git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Branch- a workspace in which you can make changes that wont affect the live site
  • Repository(repo)- a folder in which all files and their version histories are stoerd
  • code . opens VScode
    CLI SHORTCUTS:
  • pwd- displays current working directory
  • ls- displays the list of files and folders in the current directory
  • cd- navigates between directories in the path
  • cd ..- navigates back one path in your directories
  • mkdir- creates a new directory
  • touch- creates a file inside of a directory
  • start- opens a file

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • In JS, code runs from the first statement at the top of the page, down to the last statement at the bottom of the page, and the order effects the outcome with JS
  • JS items are 0 indexed, meaning the first item in an arroy is 0 not 1
  • Array- a single variable that is used to hold a group of data. They are typically related in some way
  • Index- the location of a data item in an array with a unique number
  • For loop containts the following statements: 1st- statement determines starting point, 2nd is a condition, 3rd what allows the array to iterate over each item
